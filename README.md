# 🌦️ Weather App

A simple and interactive **Weather Application** that shows real-time weather data for any city using the **OpenWeather API**.  
Built with **HTML, CSS, and JavaScript** ✨

---

## 🚀 Features
- 🔎 Search weather by city name  
- 🌡️ Display temperature, humidity, and weather condition  
- 🎨 Beautiful UI with gradient & animations  
- 💾 Save last searched city in LocalStorage  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack
- **HTML5**  
- **CSS3 (Gradient + Animations)**  
- **JavaScript (Fetch API + LocalStorage)**
