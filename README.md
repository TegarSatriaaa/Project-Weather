🌦️ Weather App

A simple and interactive weather application built with HTML, CSS, and JavaScript.
This app fetches real-time weather data using the OpenWeather API and displays:

🌡️ Current temperature

🌍 Location-based weather

🌥️ Weather conditions (sunny, cloudy, rainy, etc.)

🌬️ Additional info like humidity and wind speed

🚀 Features

Real-time weather data using OpenWeather API

Search weather by city name

Interactive and modern UI with animations

Responsive design (works on desktop & mobile)

Gradient background that changes based on weather

📦 Installation

Clone this repository

git clone https://github.com/username/weather-app.git


Open the folder

cd weather-app


Open index.html in your browser 🚀

🔑 Setup API Key

Get your free API key from OpenWeather

Open the script.js file

Replace this line with your API key:

const apiKey = "YOUR_API_KEY";
